# semj28 | Measurement Sandbox

Welcome to my personal web development and measurement sandbox. This repository houses the code for [semj28.com](https://semj28.com), a live testing ground for configuring tracking architectures and validating advanced measurement solutions.

## 🎯 Purpose
As a sandbox, this site allows for hands-on, end-to-end testing of privacy-centric measurement technologies in a controlled environment. Current and future testing focuses include:
* Enhanced Conversions
* Consent Mode 
* Google Tag Gateway
* Customer Match

## 🛠️ Tech Stack & Architecture
* **Frontend:** Vanilla HTML5, CSS3, and JavaScript
* **Hosting:** GitHub Pages
* **Proxy & Edge Logic:** Cloudflare (Free Tier)
* **Tag Management:** Google Tag Manager (GTM)
* **Domain Management:** GoDaddy

## 📂 Repository Structure
* `/` - Root directory containing primary HTML pages (index, about).
* `/assets/img/` - Image assets and media.
* `/assets/css/` - External stylesheets (coming soon).
* `/assets/js/` - Custom JavaScript and dataLayer event logic (coming soon).
